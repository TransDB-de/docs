# Architekturentscheidungen (ADRs)

ADR steht für **Architecture Decision Record** und ist ein kurzes, strukturiertes Dokument, das festhält, _warum_ eine bestimmte Architekturentscheidung getroffen wurde.
In unserem Fall benutzen wir es nicht nur für technische Entscheidungen, sondern auch für organisatorische.

!!! info "Good to know"

    Ein ADR protokolliert nicht nur _was_ entschieden wurde, sondern vor allem _warum_, damit später niemand rätseln muss, warum Dinge so sind wie sie sind.
